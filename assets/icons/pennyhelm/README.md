# PennyHelm Custom Icon Set

This folder contains a standalone custom SVG icon set for PennyHelm.

Status:
- Created for design exploration only
- Not wired into the app
- Not deployed

Design direction:
- 24x24 grid
- Rounded stroke system
- Finance-oriented geometry
- Slightly more distinctive than the current generic line icons
- Built to work with `stroke="currentColor"` so the app can theme them later

Included icons:
- `dashboard.svg`
- `bills.svg`
- `calendar.svg`
- `income.svg`
- `debts.svg`
- `accounts.svg`
- `budgets.svg`
- `savings.svg`
- `cashflow.svg`
- `rules.svg`
- `settings.svg`
- `sharing.svg`
- `taxes.svg`

Preview:
- Open `preview.html` locally in a browser to see the full set on a dark background.

Implementation notes for later:
- These are intentionally not connected to navigation or feature cards yet.
- All icons use a `24 24` viewBox and `currentColor` stroke for easy drop-in replacement.
