# PennyHelm Custom Icon Set V2

Second-pass refinement of the PennyHelm icon family.

Goal:
- Push the set closer to a premium enterprise fintech feel
- Keep the system clean, quiet, and trustworthy
- Improve metaphor clarity on the weaker first-pass icons

Design traits:
- 24x24 grid
- Rounded line icons
- More disciplined negative space
- Less decorative motion, more structural clarity
- Built with `stroke="currentColor"` for future theming

This set is standalone and not implemented in the app.
